MXenes Phonon Database is a curated set of 2D MXene materials obtained by computational screening of known MAX phases through exfoliation procedures. This database contains the relaxed 2D MXene structures and their phonon dispersion properties. For more details, please see the related publications:
Mounet et al., Nat. Nanotech. 13, 246-252, (2018) Paper DOI: 10.1038/s41565-017-0035-5
Campi et al., ACS Nano 17, 12, 11268-11278, (2023) Paper DOI: 10.1021/acsnano.2c11510
